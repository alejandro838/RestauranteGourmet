Breve descripción del sitio:
"La Buena Mesa" es un sitio web de restaurante diseñado para ofrecer a los usuarios una experiencia culinaria completa. El sitio presenta una página de inicio atractiva que introduce el concepto del restaurante y su ambiente elegante.
Cuenta con una sección de "Carta" donde se exhiben los platos principales, incluyendo descripciones detalladas, ingredientes clave y precios, permitiendo a los comensales explorar las opciones del menú.
Además, incorpora un sistema robusto de "Reservas" que facilita a los clientes registrar, actualizar o eliminar sus reservas de mesa, con una vista clara de las reservas existentes para una gestión eficiente.
El diseño general es limpio y fácil de navegar, enfocado en la usabilidad y en resaltar la oferta gastronómica.

Pasos para abrirlo localmente:
1.  Asegúrate de tener un servidor web local instalado (por ejemplo, Apache, Nginx, o utiliza la extensión Live Server de VS Code si estás trabajando con un editor).
2.  Descarga o clona todos los archivos del proyecto en el directorio raíz de tu servidor web local (por ejemplo, `htdocs` para Apache o la carpeta de tu proyecto para Live Server).
3.  Abre tu navegador web preferido (Chrome, Firefox, Edge, etc.).
4.  Escribe la URL local de tu proyecto `http://localhost/nombrecarpeta`, se reemplaza con el nombre de la carpeta por el nombre real de la carpeta donde guardaste los archivos).
5.  Si utilizas Live Server en VS Code, simplemente haz clic derecho en el archivo `index.html` y selecciona "Open with Live Server".

Cómo se publica en GitHub Pages:
1. Primero aseguramos que el archivo principal se llame index.html y sube todo el contenido al repositorio Restaurante-Gourmet.
2. Se hace clic en la pestaña "Settings" (Configuración) y luego selecciona "Pages" en el menú lateral.
3. En "Build and deployment", selecciona "Deploy from a branch" y elige la rama main (o master) y la carpeta /(root).
4. Haz clic en "Save" (Guardar) para iniciar el proceso de despliegue y espera unos minutos a que GitHub procese la publicación.
5. Vuelve a la sección "Pages" para encontrar el enlace de tu sitio web publicado, con el formato: https://[usuario].github.io/Restaurante-Gourmet/.
